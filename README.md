# ascii-earth

> THAT'S NO EARTH! — IT'S A SPACE STATION.

Рисует Землю UTF-8-постером прямо в терминале. Реальная текстура NASA Blue
Marble back-проецируется на сферу (ортографическая проекция), сэмплируется по
каждой ячейке терминала и перекрашивается в холодную синюю палитру: синий
океан, серо-белая суша, серые звёзды на чёрном, тонкая обводка и подписи.

Цветовая схема — `--palette` (по умолчанию `natural`), переключается на лету
клавишей `p`. Девять штук:

`natural` (реальные картографические цвета из текстуры — синий океан с
глубиной, зелёные леса, золото пустынь, белые льды), `political` (то же, но
ярче/контрастнее), `blue` (стилизованная сине-серая), `green` (фосфорный
CRT), `amber` (янтарный CRT), `ice` (голубой чертёж), `mono` (ч/б),
`inferno` (тепловая false-colour), `neon` (кислотный циан/маджента),
`catppuccin` (Mocha: sky-океан, teal/green леса, peach/yellow пустыни,
lavender льды).

`--sun` (или день/ночь в меню `?`) освещает планету по реальному положению
солнца на текущий момент: дневная половина яркая и цветная, ночная — **яркая
чёрно-белая** (десатурируется в серый, а не глохнет в темноту), причём ночной
океан темнее суши, так что береговые линии читаются. Планета сама не
вращается; подсолнечная точка пересчитывается каждый кадр, освещение медленно
ползёт по реальным часам (15°/час). Точка приближённая (без уравнения
времени, ~пара градусов).

`--follow` — другой режим: камера смотрит со стороны солнца, поэтому видна
только дневная сторона (полдень в центре, сумерки по ободу), а Земля
вращается под ней с реальной скоростью — **один оборот за 24 часа**, привязано
к реальному времени суток.

Три режима глифов (`--glyphs`):

- `braille` (по умолчанию) — суб-пиксельный рендер 2×4 точки на ячейку,
  почти сплошная заливка, цвет несёт сушу/море/лёд. Максимум детализации;
  смотреть в реальном терминале с Braille-глифами.
- `unicode` — плотный 64-уровневый density-ramp из UTF-8: блоки, шейды,
  геометрия, латиница/кириллица/греческий. Веса измерены в моношрифте.
- `ascii` — буквенная штриховка, ближе всего к исходному арту-постеру.

## Запуск

```sh
make install      # venv (в ~/.cache) + pillow/numpy
make run          # ИНТЕРАКТИВНО — крутить с клавиатуры
make poster       # напечатать один кадр и выйти
make spin         # авто-вращение (Ctrl-C — стоп)
```

Текстура (~240 КБ, public domain) скачивается один раз в
`~/.cache/ascii-earth/` при первом запуске. Прекэш без рендера — `make assets`.

### Интерактив (`make run` / `globe.py -i`)

Управление — минимум клавиш, всё остальное в одном меню по `?`:

| Ввод | Действие |
|---|---|
| мышь drag / стрелки / `hjkl` | вращать глобус |
| колесо / `+` `-` | зум |
| `space` | авто-вращение |
| `?` | открыть/закрыть меню настроек |
| `r` | сброс всего |
| `q` / `Esc` | выход |

В меню (`?`) одна строка-карусель: `↑`/`↓` выбирают параметр, `←`/`→`
меняют значение. Параметры: палитра, глифы, цветовой режим, день/ночь
(освещение по реальному солнцу сейчас), **follow sun** (вид со стороны
солнца — всегда видна только дневная сторона, планета медленно вращается под
ней), звёзды, обводка, подписи, насыщенность, яркость.

## Опции

```sh
python3.12 globe.py [опции]
```

| Опция | По умолчанию | Что делает |
|---|---|---|
| `-i`, `--interactive` | — | живой режим: клавиатура + мышь |
| `--glyphs {braille,unicode,ascii}` | `braille` | набор символов рендера |
| `--palette NAME` | `natural` | схема: natural/political/blue/green/amber/ice/mono/inferno/neon/catppuccin |
| `--saturation F` | `2.1` | насыщенность цветов (1 = как в текстуре) |
| `--gamma F` | `0.55` | яркость (меньше = светлее) |
| `--sun` | — | реальный терминатор день/ночь (сейчас, по UTC) |
| `--follow` | — | вид со стороны солнца: только дневная сторона, медленное вращение |
| `--spin` | — | авто-вращение |
| `--size N` | `0` (авто) | диаметр диска в столбцах; `0` = под размер терминала |
| `--lon DEG` | `-30` | центральная долгота — поворот глобуса |
| `--lat DEG` | `18` | наклон оси / широта обзора |
| `--aspect R` | `2.0` | отношение высота:ширина ячейки (правит «овал») |
| `--color {auto,256,truecolor,none}` | `auto` | цвет; `auto` берёт truecolor по `COLORTERM`, иначе 256 |
| `--ramp STR` | буквенный 48-уровневый | символы тёмное→светлое |
| `--stars F` | `0.03` | плотность звёзд `0..1` |
| `--no-stars` / `--no-ring` / `--no-labels` | — | убрать элементы |
| `--top STR` / `--bottom STR` | пасхалка | свои подписи |
| `--fps F` / `--step DEG` | `12` / `6` | скорость и шаг вращения/клавиш |

Примеры:

```sh
python3.12 globe.py -i                          # покрутить руками
python3.12 globe.py --lon 100 --lat 0           # вид на Азию, без наклона
python3.12 globe.py --color truecolor --size 110 # крупно и точная палитра
python3.12 globe.py --aspect 2.2                 # если диск выглядит овалом
python3.12 globe.py --spin --step 4 --fps 15
```

Палитра по умолчанию холодная (как на исходном арте). Под `screen`/`tmux`/`mosh`
без truecolor `auto` сам выберет 256 — для точных сине-серых тонов терминал с
truecolor предпочтительнее.

## Требования

`python3.12`, `pillow`, `numpy`. venv ставится в `~/.cache/ascii-earth/venv`
(локальный диск), а не в дерево проекта — оно отдаётся с Mac по sshfs, где venv
не собрать.
